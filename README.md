# Sistema de Estoque Distribuído
Sistema de estoque distribuído, utilizando Java RMI e Web Services.

Integrantes do projeto: Brendon Vicente Rocha Silva (18202189), Breno de Brida (18204020) e
Vítor Philip Starucka (19200442)

** O relatório do projeto, contendo uma breve explicação de
uso (compilação, inicialização, exemplo de execução do serviço
e clientes - tópicos 1.1.2 e 1.1.3), encontra-se na pasta "docs" desse diretório, com o nome "relatorio_distribuida.pdf"

De qualquer forma, para execução do sistema, posicionado na pasta do projeto:

- É só executar em seu terminal o arquivo: compile.sh

- Então rodar o servidor: server. sh

- E por fim, quantas instâncias de clientes quiser, com o arquivo: client.sh

Também é possível executar diretamente o servidor e os clientes através da sua
IDE de preferência. Configurando o PATH para o jar "json-simple-1.1.1.jar" localizado em: Sistema-Estoque-Distribuido/src/resources

No cliente, você pode executar diversas opções. Uma demonstração de uso encontra-se no relatório referenciado anteriormente **
